# Fast Parallel construction of scene BVH (LBVH) through WebGPU compute shaders.

This is my implementation of the paper "Maximizing Parallelism in the Construction of BVHs, Octrees, and k-d Trees" by Tero Karras, you can find [HERE](https://research.nvidia.com/sites/default/files/publications/karras2012hpg_paper.pdf).

You can find the demo playground of this implementation [HERE](https://skyepulse.github.io/projects/FastBVHStandalone/index.html).
